# Test de création de repo via MCP

    Ce projet est un test réalisé via un agent mcp qui dispose de divers tools et va générer un repo ou readme

    ## Getting Started

    ```bash
    git clone https://github.com/Spikesito/repo-created-with-mcp.git
    cd repo-created-with-mcp
    ```
